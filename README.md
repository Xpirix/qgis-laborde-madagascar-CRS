# Laborde CRS for each Cities in Madagascar
This repository contains QGIS custom CRS for Laborde projection in Malagasy country (Madagascar).

As we know, each city has its own projection. So, i've created a custom projection for each based on files (ie. Raster, Vector) which are on Laborde.
There are the list of available custom CRS until now :
